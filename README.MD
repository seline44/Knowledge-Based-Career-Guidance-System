# Knowledge-Based Career Guidance System
📌 Overview

The Knowledge-Based Career Guidance System is an intelligent web-based application designed to help users identify suitable career paths based on their interests, skills, and preferences.

The system uses a rule-based approach (knowledge base + inference engine) to analyze user input and generate personalized career recommendations.

🚀 Features
👤 User input for interests, skills, and preferences
🧠 Rule-based recommendation system
📚 Career database with descriptions and requirements
🔍 Search and filter career options
📊 Personalized career suggestions
💡 Educational guidance for career planning
🏗️ System Architecture

# The system follows a Knowledge-Based System (KBS) architecture:

Knowledge Base → Stores career data and rules
Inference Engine → Processes user input and applies rules
User Interface → Allows interaction with the system
🔄 Flow:
User Input → Inference Engine → Knowledge Base → Recommendations → Output
